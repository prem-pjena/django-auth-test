# Django Auth API Task

This project contains basic Signup and Login APIs built using Django and Django REST Framework.

## Features

- User Signup API
- User Login API
- Django built-in User model
- Secure password hashing using `create_user()`

## Tech Stack

- Python
- Django
- Django REST Framework
- SQLite
